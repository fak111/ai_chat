## 1. 设计概述

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| 简洁至上 | MVP 阶段专注核心功能，界面简洁清晰 |
| 一致性 | 统一的视觉语言和交互模式 |
| 即时反馈 | 所有操作都有明确的反馈 |
| 移动优先 | 针对 Android 移动端优化 |
| 无障碍 | 合理的触摸区域和对比度 |

### 1.2 页面总览

| 页面ID | 页面名称 | 描述 | 对应功能 | 优先级 |
